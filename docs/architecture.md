## Communication

- Le **renderer Vue** ne contacte jamais l'API directement.
- Les appels passent par **IPC** (`preload.js` вҶ’ `main.js` вҶ’ `apiClient`).
- Le main process exГ©cute les requГӘtes HTTP vers `API_BASE_URL`.
